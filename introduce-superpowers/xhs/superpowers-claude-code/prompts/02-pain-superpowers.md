# Xiaohongshu Content Card — Notion Style (Comparison Layout)

Create a Xiaohongshu content card in Notion minimalist hand-drawn style. Match the visual style of the reference image (cover card).

## Canvas
- Aspect ratio: 3:4 (portrait)
- Background: warm off-white #FAFAFA

## Layout: Comparison
Two columns: Problems (left, red X marks) vs Solution (right, green checkmarks).

## Content Structure

### Title (top)
Large bold text: 你有没有这种痛苦？
Small subtitle: 用 Claude Code 编程时的三大坑

### Left Column — BEFORE (Problems)
Header: ❌ 以前
- 写完跑偏，改，再跑偏
- Plan Mode 执行阶段临时发挥
- AI 说"完成了"，其实没验证

### Right Column — AFTER (Solution)
Header: ✅ 用了 Superpowers
- Skills 锁定方向，不跑偏
- 每步有检查点，及时纠偏
- 强制验证后才能宣布完成

### Bottom Tag
小字: 安装 Superpowers，14 个 Skills 自动激活

## Style Details
- Match the reference image style: hand-drawn stick figures, wobble lines, off-white background
- Left column background: very light pink/red tint
- Right column background: very light green tint
- Red X marks hand-drawn style
- Green checkmarks hand-drawn style
- Same charcoal ink #37352F for all text
- Orange #E9963E accent on "Superpowers" text
- Hand-drawn dividing line between columns
- No watermark
