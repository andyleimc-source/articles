# Xiaohongshu Cover — Notion Style

Create a Xiaohongshu (Little Red Book) cover image in Notion minimalist hand-drawn style.

## Canvas
- Aspect ratio: 3:4 (portrait)
- Background: warm off-white #FAFAFA with subtle paper texture

## Layout: Sparse
Large visual impact. Main illustration takes center stage with generous whitespace.

## Visual Elements
- Center: A simple hand-drawn stick figure / cartoon character sitting at a laptop, wearing a superhero cape in orange (#E9963E)
- Around the character: 4-5 small floating cards (hand-drawn rectangles) labeled: "brainstorming", "TDD", "code review", "verify", "planning" — in clean sans-serif
- Top area: Large bold title text
- Bottom area: Subtitle text

## Typography
- Main title (large, bold, black #1A1A1A): 用了 Superpowers
- Sub-title line 2 (large, bold): 我的 Claude Code
- Sub-title line 3 (large, bold): 返工少了九成
- Small tag below: 14 个 Skills 覆盖开发全流程

## Style Details
- Notion minimalist: clean white/off-white background
- Hand-drawn wobble effect on all illustrations
- Single-weight ink lines, dark charcoal #37352F
- Orange accent #E9963E for the superhero cape/highlight only
- Pastel yellow highlight on title text
- Small hand-drawn circle-mark around "九成" (90%)
- Generous whitespace, uncluttered
- No watermark

## Overall Feel
Intellectual, clean, slightly playful — like a Notion page that became a poster.
