# 这个号的爆款标题 DNA（必学）

下面是这个号在微信"看一看"推荐池里拿到 6000+ 阅读的标题，请你提炼共性：

- 「我把 Claude Code 塞进了微信，从此可以躺在床上写代码」(20847 阅读)
- 「用了 Superpowers，我的 Claude Code 返工少了九成」(11686 阅读)
- 「Claude Code 隐藏技巧新手必看」(6867 阅读)
- 「我把整台 Mac 塞进了 iPad，从此电脑关机也能 vibe coding」(6210 阅读)
- 「我给自己的 Mac 装了"过目不忘"，然后让 Claude Code 替我回忆」(3339 阅读)

## 共性规律
1. **第一人称 + 具体动作动词**："我把 X 塞进了 Y"、"我给自己装了 X"、"用了 X，我的 Y 少了九成"
2. **反直觉钩子**：把"重东西"塞进"轻容器"（CC 塞进微信、Mac 塞进 iPad）、把"差结果"变成"具体数字"（返工少九成）
3. **场景画面**：让读者立刻看见自己也能这样做的画面（"躺在床上写代码"、"电脑关机也能 vibe coding"、"替我回忆"）
4. **避开**：纯技术名词堆叠、抽象观点宣言、"完整指南/全攻略"这种没人物没动作的套话
5. **字数**：14-28 字最佳。微信推荐池在手机上显示 2 行最多 28 字

## 反例（低阅读 < 100）
- "AI 已经帮你打掉了 80% 的工作，剩下 20% 决定你的位置"（44 阅读）—— 抽象、无场景、第二人称指责感
- "重新定义智能：从「会预测」到「会行动」"（9 阅读）—— 学术腔、无动作、无人物
