# Terminal Starter Guide 终端入门指南

> 专为 Mac 新手设计的终端工具全面指南 / A comprehensive terminal guide designed for Mac beginners

---

## 这是什么 / What is This?

本项目帮助你从零开始掌握 Mac 终端，涵盖基础概念、常用命令、效率工具和版本管理。  
This guide helps you master the Mac terminal from scratch — concepts, commands, productivity tools, and version control.

**学完之后你可以 / After reading, you will be able to:**

- 理解终端、Shell 等核心概念，不再感到陌生
- 熟练使用几十个日常必备命令
- 配置好看又高效的 iTerm2 终端环境
- 用 `z` 秒跳任意目录，告别反复 `cd`
- 用 `fzf` 模糊搜索历史命令、文件和目录
- 用 Git + GitHub 管理和分享自己的代码

## 环境要求 / Requirements

- macOS（任意近代版本）
- [Homebrew](https://brew.sh) 包管理器（大多数工具通过它安装）

安装 Homebrew / Install Homebrew:

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
