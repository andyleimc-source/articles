# 内容分析

## 主题
《皮册寓言》——用 5 个寓言故事讲透 Claude Code 项目文档协作体系

## 类型
知识科普 / 故事化干货 / 工作流分享

## 核心要点
1. Claude 每次新会话都失忆 → 需要外部"皮册"承载记忆
2. 一套 6 个 md 文件：CLAUDE.md / plan.md / progress.md / decision.md / bug.md / handoff.md
3. CLAUDE.md 两条铁律：≤200 行 + 三层分层
4. decision.md 升级路径：单文件 → docs/adr/ ADR
5. subagent 上下文隔离 → 必须事前写明
6. 文档维护三纪律：触发即更 / 真实日期 / 改完 commit

## 受众
深度使用 Claude Code 的开发者 / AI Agent 协作探索者 / 个人开发者 + 小团队 lead

## 钩子潜力
高 —— 标题"Claude 很聪明，但它会失忆"自带认知冲突 + 寓言"国王 + 失忆智者"画面感强

## 视觉机会
5 个故事全部为古代寓言场景（国王/谋士/石碑/商人/镖师/灯塔），天然具备绘本插画基底

## 推荐方案
- 策略：A（故事驱动 / Story-Driven）—— 文章本身就是寓言叙事
- 风格：自定义 `warm-storybook-fable`（基于 retro / warm 派生）
- 布局：cover=sparse；故事页=balanced；金句页=sparse
- 配色：羊皮纸暖色 + 局部水彩点染
- 张数：7（已用户确认）
- 比例：3:4

## 语言
中文（zh-CN）
