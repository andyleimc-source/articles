## 正文（约 220 字，可直接复制）

用 Claude Code 久了你会发现一件让人崩溃的事——

它真的很聪明，但**每次新会话都失忆**🤯
你昨天讲过的来龙去脉、为什么选 A 不选 B、那个 bug 的根因，它一概不知道。每天重新讲一遍，讲到自己都烦。

我摸索了半年，总结了一套轻量办法：给 Claude 铺一本"皮册"📜

不是一本，是**一套 6 个 md 文件**：
✦ CLAUDE.md（项目说明书）
✦ plan.md（当前目标）
✦ progress.md（流水账）
✦ decision.md（架构决策）
✦ bug.md（已知问题）
✦ handoff.md（下轮快照）

讲了 5 个寓言故事——
🏰 健忘的智者
🪨 律法石碑
🏪 商人一桩一单
🐎 三路镖师
🗼 守灯塔的老人

每个故事对应一条具体做法。读完，你也能让你的"AI 谋士"从此不再失忆。
