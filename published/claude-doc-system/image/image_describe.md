# 图片说明

整体风格统一为 Notion.so 扁平卡通插画（Seedream 生成），柔和米色背景、手绘不规则线条、无阴影。

## cover.png（文章顶部主视觉）
一本打开的棕色皮册居中，四周漂浮五个小图标：金色王冠、石碑、红印单据、马队剪影、小灯塔——暗示文章由五个寓言构成。
- **论点关联**：为"皮册寓言集"定调

## image/fable-scene.png（故事一内）
三个卡通人物：戴王冠的国王、捧皮册的老工匠、戴方帽的谋士。
- **论点关联**：寓言一的核心画面——皮册的诞生

## image/modern-notebook.png（故事一揭示段）
俯视桌面：手写清单笔记本 + 休眠笔记本电脑（飘着 zzz）+ 咖啡、钢笔、便利贴。
- **论点关联**："当笔记本电脑睡了，你的笔记本不能睡"——把古代皮册映射到现代

## image/six-files.png（故事一揭示段结尾）
中心是戴眼镜的 Claude 机器人头像（失忆的智者），6 张带图标的文件卡片环绕：CLAUDE.md / plan / progress / decision / bug / handoff，每张卡片大号中文标签。
- **论点关联**：六文件皮册全景

## image/stone-stele.png（故事二内）
戴金冠红袍的卡通国王站在大石碑前仰视；两侧各三栋中式府衙建筑，门口挂着小册子。
- **论点关联**：律法石碑寓言 + CLAUDE.md 分层布局

## image/claude-md-layers.png（故事二揭示段）
三层叠放卡片图：全局偏好（小房子）/ 项目根目录（文件夹）/ 子目录（三个小卡片，懒加载），右侧向上箭头"向上合并"，纯中文大号标签。
- **论点关联**：CLAUDE.md 三层 walk-up + 子目录懒加载机制

## image/merchant-receipts.png（故事三内）
白胡子老商人坐在柜台后，柜台上一个打开的抽屉装满盖红印的单据，旁边一本摊开的竖栏账册。
- **论点关联**：一桩生意一张单——对应 ADR 每决策一文件

## image/three-caravans.png（故事四内）
地图俯视：三条蜿蜒小路，分别有红/蓝/绿三队骑马镖师；前景一位总镖头捧着卷起的路书。
- **论点关联**：subagent 上下文隔离——能写在纸上的不靠临时传话

## image/lighthouse-keeper.png（故事五内）
夜空下灯塔剖面，塔顶亮着黄灯，戴毛线帽的老人在小木桌前用鹅毛笔写日志，灯塔伸出光束照向海面。
- **论点关联**：触发即更、日期真实——灯熄即点，事发即记
