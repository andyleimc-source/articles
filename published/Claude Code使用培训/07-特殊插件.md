# 第七阶段：特殊插件

## 7.1 Claude use Chrome

### 什么是 Claude use Chrome

Claude use Chrome 是通过 **Chrome CDP（Chrome DevTools Protocol）** 协议控制浏览器的能力。简单说，就是让 Claude 能够操作 Chrome 浏览器。

### CDP 协议原理

```
Claude ←→ CDP Server ←→ Chrome Browser
         (MCP 或直接连接)
```

Chrome 内置了 CDP 服务器，通过它可以：
- 获取页面内容
- 点击元素
- 填写表单
- 截取截图
- 执行 JavaScript

### 使用前提

```bash
# 1. 安装 Chrome
#    下载地址：https://www.google.com/chrome/

# 2. 开启 Chrome 远程调试模式
#    Mac 上启动带调试的 Chrome：
open -a "Google Chrome" --args --remote-debugging-port=9222

#    Windows 上：
#    "C:\Program Files\Google\Chrome\Application\chrome.exe" --remote-debugging-port=9222
```

### 典型使用场景

| 场景 | 说明 |
|------|------|
| 自动填表 | 打开网页、自动填写表单、提交 |
| 网页抓取 | 获取动态加载的内容 |
| UI 测试 | 自动化测试网页功能 |
| 批量操作 | 批量处理重复性网页操作 |

### 安全提示

> ⚠️ Chrome 控制涉及隐私：
> - 只在可信环境下使用
> - 避免在登录状态下控制他人网页
> - 敏感操作前确认 URL

## 7.2 Claude use Desktop

### 什么是 Claude use Desktop

Claude use Desktop 让 Claude 能够控制 macOS 桌面应用和系统级操作。

### 支持能力

| 能力 | 说明 |
|------|------|
| 应用控制 | 打开、关闭、操作应用 |
| 系统自动化 | 执行 AppleScript、快捷指令 |
| 截图 | 捕获屏幕内容 |
| 文件操作 | 创建、移动、删除文件 |
| 剪贴板 | 读写剪贴板 |

### macOS 辅助功能权限

要控制桌面应用，需要开启辅助功能权限：

```
系统偏好设置 → 安全性与隐私 → 隐私 → 辅助功能
```

**配置步骤：**
1. 打开「系统偏好设置」
2. 进入「安全性与隐私」
3. 点击「隐私」标签
4. 向下滚动找到「辅助功能」
5. 点击左下角 🔒 解锁
6. 添加 Terminal（或 Claude Code 应用）

### 典型使用场景

| 场景 | 说明 |
|------|------|
| 自动操作 GUI 应用 | 代替鼠标点击 |
| 截图并分析 | 截屏发给 Claude 分析 |
| 读写剪贴板 | 快速处理剪贴板内容 |
| 快捷指令 | 触发 macOS Shortcuts |

### 使用示例

```python
# 通过 AppleScript 控制应用
# 打开 Finder
osascript -e 'tell application "Finder" to activate'

# 通过 Peekaboo Skill 截图
# 参考 ~/.agents/skills/peekaboo/SKILL.md
```

### 安全提示

> ⚠️ 桌面控制权限敏感：
> - 恶意使用可以监控键盘输入（记录密码等）
> - 只在可信环境使用
> - 定期检查已授权的应用列表
> - 使用完毕后考虑关闭权限
