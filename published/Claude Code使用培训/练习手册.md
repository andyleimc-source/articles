## 参考答案

### Q1.1 参考答案

> Claude Code 最核心的区别是：**它能直接操作本地文件系统并执行终端命令**。
> ChatGPT Web 版只能"聊天"，不能读写你的文件、不能跑脚本、不能调用本地工具。
> 本质上 Claude Code 是"AI + 文件系统 + 命令行"，而 ChatGPT 是纯对话。

### Q3.1 参考答案

> MCP（Model Context Protocol）是一种协议标准，让 AI 模型能连接外部工具和数据源。
> 简单说：AI 模型本身只会文本对话，通过 MCP 它能"长出手脚"，操作文件、调用 API、控制应用。
> 类似于 USB 接口——电脑通过 USB 可以连接打印机、硬盘、摄像头；AI 通过 MCP 可以连接 GitHub、邮件系统、日历等。

## 附录：常见错误排查

| 错误 | 解决方法 |
|------|---------|
| `command not found: claude` | npm 安装失败，重新执行 `npm install -g @anthropic-ai/claude-code` |
| `curl: (7) Failed to connect` | 网络问题，需要翻墙 |
| `permission denied` (git) | 未配置 Git 用户名和邮箱，执行 `git config` |
| GitHub push 被拒绝 | SSH 密钥未配置或未添加到 GitHub |
| Tailscale 无法连接 | 两端都需要运行 Tailscale 并登录同一账号 |
