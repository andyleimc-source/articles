# 封面图 v2：减法时代 · 七条手册

一张电影感静物概念图，**16:9 比例**，editorial 摄影质感，深色木质书桌 flat-lay 俯视构图，温暖琥珀色侧光。整体氛围与正文里的"四代纸张演化"图保持视觉一致（同一桌面、同一光线、同一风格）。

**核心隐喻：** 一本厚重过时的"规则大典"被合上、推到一边，旁边新摆出一本极薄、克制、新派的"七条手册"——象征 prompt 工程从"堆规则"进入"减法时代"。

**画面构图：**

**左侧（暗一点的区域）：**
- 一本厚重的旧式精装书，**非常厚**（书脊 8-10 cm 那种），皮质封面已陈旧、带磨损边角
- 书是合上的，**斜向放置且部分被推出画面边缘**（暗示"已经过时了"）
- 封面上有一行凹刻的英文小字：`The Old Prompt Stack`（这是允许的少量文字之一，必须清晰可读）
- 书边夹着几张露出来的泛黄索引卡，模糊可见 `ALWAYS` / `NEVER` 之类的大写词残片（不要清晰整段文字，只是模糊残片即可）
- 一支老旧钢笔放在书旁，象征"作者放下笔了"

**右侧（亮一点的区域，是视觉重心）：**
- 一本**极薄**的现代手工装订册子，米白色或奶油色封面，简洁优雅
- 封面正中央有一枚**手工压印的烫金圆形印章**，印章里只刻一个数字 `7`，简洁有力
- 印章下方一行优雅的英文小字：`Prompt 5.5 · A New Playbook`
- 印章上方一行更小的中文小字：`减法时代`（手写体或宋体衬线，字形必须清晰锐利）
- 册子下方斜放一支细黑色现代设计感钢笔（与左侧旧钢笔形成对比）

**桌面氛围：**
- 深胡桃木色桌面，纹理自然
- 暖琥珀色斜光从画面右上方照下来，投下柔和阴影
- 左半部分稍暗、偏冷灰；右半部分稍亮、偏暖琥珀——视觉引导从左向右推进
- 留白充足，物品周围有大量呼吸空间

**绝对禁止：**
- 不要任何屏幕、电脑、键盘、芯片、机器人元素
- 不要任何代码块、伪代码、整段英文文本
- 不要紫色霓虹渐变、cyber 风格、3D 卡通
- 不要堆叠的圆角卡片、generic hero、装饰性渐变
- 除了上面明确指定的几处文字（`The Old Prompt Stack`、`7`、`Prompt 5.5 · A New Playbook`、`减法时代`）以外，画面里**不允许出现任何其他文字**——所有文字必须字形清晰、无错字、无乱码

**风格关键词：** still life photography、editorial cover、museum object aesthetic、warm amber side light、aged leather + cream paper texture、minimalist composition、cinematic depth of field（左侧旧书略带 bokeh，右侧手册清晰锐利）。

**核心要传达的情绪：** 一种"翻篇"的从容感——旧规则被悄悄合上，新手册从容登场。安静、克制、有重量。
