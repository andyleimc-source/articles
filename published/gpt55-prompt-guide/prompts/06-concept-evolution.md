# 概念图：四代 prompt 的"厚度演化"

一张极简的概念静物摄影风格图像，16:9 比例，俯视视角（top-down flat lay），深色木质桌面背景。

**画面主体：** 桌面上从左到右依次摆放四件物品，呈现"使用者负担从厚到薄"的递减序列。物品风格统一、纸张材质质感真实，每件物品左侧或下方有一个细小的金属铭牌或浮雕年份数字（年份就够了，不要其他文字）。

**最左：一摞 200 页的厚装订手册**
- 牛皮纸封面已泛黄，边缘有翻阅痕迹
- 像 90 年代 IBM 或 Nokia 那种说明书的厚度
- 旁边放一支铅笔
- 铭牌：`2021`

**左中：一本中等厚度的活页夹**
- 透明塑料封面，里面装着印满 XML 标签的打印纸
- 厚度约为左侧手册的 60%
- 铭牌：`2023`

**右中：一摞 A4 打印纸**
- 用回形针夹着，更薄
- 隐约可见红色印章和高亮重点
- 厚度约为最左侧的 30%
- 铭牌：`2024`

**最右：一张黄色便利贴**
- 单独躺在桌面上
- 极薄、有一点轻微卷边
- 上面只有一行手写字 `Translate to French.`（这是整张图唯一的文字内容，必须清晰可读）
- 铭牌：`2026`

**整体氛围：**
- 深色木质桌面，自然光从左上角洒下，温暖琥珀色光线
- 像一张博物馆展柜的静物摄影，或一个匠人收藏家的工作台
- 物品之间的负空间很大，留白充足
- 颜色从左到右整体趋势：泛黄陈旧 → 灰白干净 → 鲜亮简洁
- 极少装饰，没有任何机器人、人物、电脑、屏幕

**严格禁止：**
- 不要紫色渐变、霓虹光、3D 渲染感
- 不要任何代码块图标、AI 大脑、芯片
- 不要任何看起来"很 AI"的视觉俗套
- 除了便利贴上那行 "Translate to French." 之外，其他物品上不要出现可读的英文段落或代码

**风格关键词：** flat-lay 静物摄影、博物馆质感、极简、温暖光线、纸张/木材的真实材质、editorial photography。

**核心隐喻：** prompt 工程的演化 = 工程师案头资料的"厚度退化"，从厚书到便利贴，象征模型变强后使用者负担减轻。
