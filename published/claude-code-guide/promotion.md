**文案（口播/字幕稿）：**

很多人问我，Claude Code 怎么装都装不上？

其实问题不在 Claude Code，在于你底层没配好。

四层基础设施，缺一不可：

第一，梯子。别用免费的，推荐 Shadowsocks，年付 20 美金，节点选美国或日本。

第二，代理软件。用 Clash Verge，一定要开 TUN 模式。不开这个，终端里的请求根本不走代理，所以你开了梯子还是报地区限制。

第三，账号。注册需要海外手机号，接码平台 hero-sms.com，两美金就能搞定。手机 App 要美区账号，银河录像局买一个，20 块钱。

第四，支付。国内卡订阅大概率被拒，买礼品卡最稳。

配好这四层，不只是 Claude Code，GPT、Gemini、DeepSeek 全都能用。

配一次，受益所有 AI 工具。
