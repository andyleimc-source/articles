**与文章的关系**：对应第一节，视觉化"盯盘人"概念——同事用 Claude Code 做了销售自动化系统后的状态。

## 02-industrial.jpg

## ⚠️ 图片压缩记录（2026-04-08）

**问题**：微信公众号编辑器通过剪贴板粘贴图片时，文件过大导致静默失败（图片显示不出来，无报错）。

**受影响图片**：02-industrial.jpg（原 4.7MB）、04-ai-wave.jpg（原 2.9MB），以及其余 AI 生成图（2.3–3.1MB）。

**根本原因**：微信剪贴板粘贴图片上限约 1MB，超过则静默失败。

**修复**：用 `sips` / `ffmpeg` 将所有图片压缩到长边 ≤ 1200px，结果 60K–424K，全部通过。

**命令**：
```bash
# jpg
sips --resampleHeightWidthMax 1200 image.jpg --out image.jpg
# webp
ffmpeg -i image.webp -vf "scale='if(gt(iw,ih),1200,-2)':'if(gt(iw,ih),-2,1200)'" -quality 85 -y image.webp
```
