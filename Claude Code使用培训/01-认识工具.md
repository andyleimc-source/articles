# 第一阶段：认识工具

## 1.1 Claude Code 是什么

### 官方定位

Claude Code 是 **Anthropic 官方推出的命令行工具（CLI）**，直接与 Claude 对话、执行代码操作、管理文件和项目。它不是网页版 Chat，也不是 IDE 插件——它是独立的命令行程序，专注**本地开发场景**。

### Claude Code 能做什么

| 功能 | 说明 |
|------|------|
| 读写文件 | 读取、创建、编辑本地文件 |
| 执行命令 | 在终端运行 shell 命令 |
| 浏览代码库 | 理解项目结构、搜索代码 |
| 调用工具 | 通过 Skills 和 MCP 扩展能力 |
| 持久化上下文 | 跨会话保持记忆和项目理解 |
| 自主操作 | bypass permission 下可控制电脑 |

### 与 ChatGPT Web 版的核心差异

| 对比项 | Claude Code | ChatGPT Web |
|-------|-----------|------------|
| 交互方式 | 终端/CLI | 网页 |
| 文件操作 | 直接读写本地文件 | 不能直接操作 |
| 命令执行 | 直接运行系统命令 | 不能 |
| MCP 扩展 | ✅ 支持 | ❌ 不支持 |
| 持久化 | ✅ 通过 workspace | ❌ 每次新建 |
| 使用场景 | 开发工作流 | 问答、写作 |

> 💡 **一句话总结**：ChatGPT Web 版是"聊天机器人"，Claude Code 是"AI 开发助手"。

### 安装 Claude Code

```bash
# macOS / Linux
npm install -g @anthropic-ai/claude-code

# 验证安装
claude --version
```

## 1.2 课前必读

> ⚠️ **国内用户必看**：以下内容是顺利使用 Claude Code 的前置条件，请先完成这些准备。

### 必做准备清单

| 准备项 | 说明 |
|--------|------|
| ☁️ 海外账号 | Gmail / GitHub 账号（注册需要海外手机号）|
| 🌐 翻墙网络 | 必须能访问 api.anthropic.com |
| 💳 充值方式 | 海外信用卡 或 充值卡 |
| 📱 验证码 | 注册时需要接收短信验证码 |

### 海外手机号获取方案

| 方案 | 平台推荐 | 费用 | 备注 |
|------|---------|------|------|
| 虚拟接码平台 | TextNow / ReceiveSMSOnline | 免费/低价 | 常用平台 |
| 实体卡 | 自行购买 | 较高 | 长期使用推荐 |

### 充值方式

| 方式 | 说明 |
|------|------|
| 海外信用卡 | Visa / Mastercard，直接支付 |
| 充值卡 | 第三方平台购买兑换码 |

### API 额度说明

Claude Code 默认使用 Claude Web 的对话额度。如果需要更高用量，可以通过 API Key 方式接入（需要充值）。
