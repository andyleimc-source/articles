## 9.1 海外账号体系

### 为什么需要海外账号

Claude Code 依赖海外服务，需要以下海外账号：

| 平台 | 用途 | 必要性 |
|------|------|--------|
| **Gmail** | 注册 GitHub、接收验证码 | ⭐⭐⭐ 必填 |
| **GitHub** | 使用 Claude Code CLI、托管代码 | ⭐⭐⭐ 必填 |
| **美区 Apple ID** | 下载海外 App Store 应用 | ⭐⭐ 可选 |

### Gmail 注册要点

**注意事项：**
- 注册时需要**海外手机号**接收验证码
- 国内手机号无法完成验证
- 推荐使用虚拟手机号（见 9.2）

**注册步骤：**
1. 访问 https://gmail.com
2. 点击「创建账号」
3. 填写信息（姓名、用户名、密码）
4. 手机验证（必须）
5. 备用邮箱（可选）
6. 完成注册

## 9.2 美区账号注册进阶

### 虚拟手机号获取

| 平台 | 网址 | 费用 | 稳定性 |
|------|------|------|--------|
| TextNow | textnow.com | 免费（带广告）| 一般 |
| TextFree | textfree.us | 免费 | 一般 |
| ReceiveSMSOnline | receivesmsonline.com | 免费 | 不稳定 |
| 超级SIM | chuangtasim.com | 付费 | 较高 |

**推荐流程：**
1. 注册 TextNow 账号
2. 获取一个美国号码
3. 用这个号码注册 Gmail / GitHub

### 充值卡购买

| 渠道 | 特点 |
|------|------|
| 淘宝/闲鱼 | 方便，但有风险（黑卡）|
| 美国官网 | 正规，但需要美国支付方式 |
| 虚拟卡平台 | Depay、Nobepay 等 |

**淘宝购买注意：**
- 选择信誉好的卖家
- 查看评论和销量
- 询问是否支持退款

### 验证码收取平台

当网站只接受短信验证时：

1. 访问接码平台（如 TextNow）
2. 获取一个临时号码
3. 在目标网站输入该号码
4. 在接码平台查看收到的验证码
5. 完成验证

**注意事项：**
- 号码有时效性，长时间不用会被回收
- 部分平台号码已被大量使用，可能被某些网站封禁
- 敏感账号建议使用长期有效的号码

### 美区 Apple ID 注册

**方法一：网页注册（推荐）**
1. 访问 https://appleid.apple.com
2. 点击「创建 Apple ID」
3. 国家选择「美国」
4. 填写信息，邮箱和手机验证
5. 完成

**方法二：手机注册**
1. 退出 App Store 现有登录
2. 随便找一个免费 App 点击下载
3. 系统提示注册时按步骤创建
4. 地区选择「美国」

**注册后设置：**
- 登录 https://reportaproblem.apple.com
- 设置付款方式为「无」
- 填写美国地址（可用免税州如俄勒冈）
- 之后可以下载美区独占 App

### 避坑指南

| 坑 | 避坑方法 |
|----|---------|
| 账号被封 | 不要短时间内大量操作 |
| 充值不到账 | 选择有保障的平台 |
| 验证码收不到 | 多试几个号码 |
| 账号被关联 | 避免同一 IP 注册多个账号 |
| 支付失败 | 确保支付方式支持外币 |
