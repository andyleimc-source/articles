# 第二阶段：环境准备

## 2.1 必装环境

### Homebrew — macOS 包管理器

> Homebrew 是 macOS 上最常用的包管理器，安装其他工具前必须先装它。

```bash
# 安装 Homebrew（终端中运行）
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 安装完成后配置 PATH（Apple Silicon Mac 需要）
echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile
eval "$(/opt/homebrew/bin/brew shellenv)"

# 验证
brew --version
```

### Git — 版本控制

```bash
brew install git

# 配置身份（每次提交需要）
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"
```

### Node.js — npm 包管理器基础

```bash
brew install node

# 验证
node --version
npm --version
```

### Python — pip 包管理器

```bash
# 推荐用 pyenv 管理多版本
brew install pyenv

# 配置 pyenv（在 zshrc 中添加）
echo 'eval "$(pyenv init -)"' >> ~/.zshrc
source ~/.zshrc

# 安装 Python
pyenv install 3.11.0
pyenv global 3.11.0

# 验证
python --version
pip --version
```

### 常用工具一键安装脚本

```bash
# 一次性安装所有核心工具
brew install git node python pyenv
```

## 2.3 翻墙与网络

### 方案对比

| 方案 | 工具 | 特点 |
|------|------|------|
| 传统 | Shadowsocks | 配置稍复杂，机场多 |
| 现代 | ClashVerge | 界面友好，规则灵活 |
| 商业 | NordVPN / ExpressVPN | 即开即用，稳定 |

### ClashVerge 配置示例

```bash
# 安装
brew install --cask clash-verge

# 使用
# 1. 获取机场订阅链接（从机场提供商处获取）
# 2. 打开 ClashVerge → 配置 → 订阅 → 添加订阅链接
# 3. 选择节点，启动代理
```

### 验证网络连通性

```bash
# 测试能否访问 Anthropic
curl -s https://api.anthropic.com/v1/models | head

# 如果返回 JSON 说明网络正常
```
