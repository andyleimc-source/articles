# 第八阶段：远程使用

## 8.1 Tailscale + Termius 远程方案

### 背景需求

有时候本地机器性能不够，或者需要访问特定网络资源，需要**在远程服务器上运行 Claude Code**。

典型场景：
- 公司服务器有独立网络，能访问内网系统
- 服务器 GPU 性能强，适合跑 AI 任务
- 需要24小时运行的 AI 服务

### 方案架构

```
┌──────────────┐         ┌──────────────────┐
│  本地电脑      │ ──────→ │  远程服务器       │
│              │  Tailscale │  (已安装 Claude  │
│  (Termius)   │  内网穿透  │   Code)          │
└──────────────┘         └──────────────────┘
       ↓
   SSH 连接
```

## Tailscale 内网穿透

### 什么是 Tailscale

Tailscale 是一款基于 WireGuard 协议的**零配置内网穿透工具**。它能让不同网络的设备安全地组成虚拟局域网。

### Tailscale 核心概念

| 概念 | 说明 |
|------|------|
| **MagicDNS** | 给每台设备分配固定域名 |
| **Tailnet** | 所有设备组成的虚拟网络 |
| **Funnel** | 将本地服务暴露到互联网（可选）|
| **Exit Node** | 让其他设备通过 Tailscale 节点上网 |

### 安装 Tailscale

```bash
# macOS
brew install --cask tailscale

# Linux (Ubuntu/Debian)
curl -fsSL https://tailscale.com/install.sh | sh

# 启动并登录
tailscale up
# 会自动打开浏览器完成 GitHub/Google 账号授权
```

### 查看设备列表

```bash
# 查看当前 Tailnet 上的所有设备
tailscale status

# 示例输出：
# 100.64.1.2   macbook-pro    user@github
# 100.64.1.3   ubuntu-server  user@github
# 100.64.1.4   iphone         user@github
```

### 获取设备 IP

```bash
# 服务器 IP 格式：100.64.x.x
tailscale status | grep ubuntu-server

# 记住这个 IP，例如：100.64.1.3
```

## Termius 远程终端

### 什么是 Termius

Termius 是一款跨平台 SSH 客户端，支持 Mac、Windows、Linux、手机端。

**优点：**
- 界面友好
- 支持 SSH 密钥管理
- 会话同步（跨设备）
- 支持 SFTP

### Termius 下载

| 平台 | 地址 |
|------|------|
| macOS | App Store 或官网 |
| Windows | 官网下载 |
| iOS/Android | 应用商店 |
| Linux | 官网或 snap |

官网：https://termius.com/

### SSH 连接配置

**步骤一：添加服务器**

1. 打开 Termius
2. 点击 **New Host**
3. 填写信息：

```
Hostname: 100.64.1.3        ← Tailscale IP
Username: ubuntu            ← 服务器用户名
Port: 22                    ← 默认 SSH 端口
```

**步骤二：配置认证方式**

**方式 A：密码**
```
Password: your_password
```

**方式 B：SSH 密钥（推荐）**

```bash
# 本地生成密钥对
ssh-keygen -t ed25519

# 将公钥复制到服务器
ssh-copy-id ubuntu@100.64.1.3

# 在 Termius 的 Keychain 中添加私钥
```

**步骤三：连接**

1. 选择主机
2. 点击 Connect
3. 首次连接需要确认指纹（yes）
4. 输入密码（或使用密钥）

## 完整使用流程

### 第一步：确保两边都登录 Tailscale

```bash
# 本地 Mac
tailscale status

# 远程服务器
tailscale status
```

### 第二步：确认能互相 ping 通

```bash
# 在本地 ping 服务器
ping 100.64.1.3
```

### 第三步：用 Termius 连接

1. Termius → 选择主机 → Connect
2. 连接成功后，看到服务器终端提示符

### 第四步：在服务器上安装 Claude Code

```bash
# 在远程服务器上执行
npm install -g @anthropic-ai/claude-code

# 验证
claude --version
```

### 第五步：正常使用

```bash
# 启动 Claude Code
claude

# 指定工作目录
claude /home/ubuntu/project
```

## 进阶：安全加固

### 使用 SSH 配置文件简化连接

```bash
# 编辑 ~/.ssh/config
nano ~/.ssh/config

# 添加以下内容：
Host server
    HostName 100.64.1.3
    User ubuntu
    IdentityFile ~/.ssh/id_ed25519
    Port 22

# 保存后直接用别名连接
ssh server
```

### Tailscale HTTPS 功能（可选）

```bash
# 开启 HTTPS（让本地浏览器访问服务器上的网页服务）
tailscale serve --bg

# 生成 HTTPS 地址
tailscale serve status
```

### 防火墙配置

```bash
# 只允许从 Tailscale IP 连接 SSH
sudo ufw allow from 100.64.0.0/10 to any port 22
sudo ufw enable
```

## 故障排查

| 问题 | 解决方法 |
|------|---------|
| ping 不通 | 检查 Tailscale 是否都在线、状态是否为 Active |
| SSH 连接超时 | 检查服务器 SSH 服务是否运行 `sudo systemctl status sshd` |
| 密码错误 | 确认用户名和密码正确 |
| 密钥登录失败 | 检查 `~/.ssh/authorized_keys` 是否有公钥内容 |
| Tailscale 无法登录 | 检查网络是否能访问互联网 |
