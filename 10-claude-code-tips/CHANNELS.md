# 社交媒体发布规范

## 抖音

| 项目 | 规范 |
|------|------|
| 标题字数 | ≤ 20 字（含标点） |
| 描述字数 | ≤ 1000 字（含标点） |
| 图片模式 | 竖版 3:4（1080×1440px），多图最多 35 张 |
| 发布要求 | 标题吸引眼球，首句制造钩子；正文用短句分行，节奏感强；结尾引导互动（评论/收藏）；话题标签 3-5 个，放正文末尾 |

---

## 小红书

| 项目 | 规范 |
|------|------|
| 标题字数 | ≤ 20 字（含标点） |
| 描述字数 | ≤ 1000 字（含标点） |
| 图片模式 | 竖版 3:4（1080×1440px），封面图最重要，系列图 2-10 张 |
| 发布要求 | 标题含关键词，带数字或情绪词更易点击；正文真实感叙事，分段清晰；emoji 适量点缀；话题标签 5-10 个，放末尾 |

---

## 微信公众号

| 项目 | 规范 |
|------|------|
| 标题字数 | ≤ 20 字（含标点） |
| 描述字数 | ≤ 1000 字（含标点） |
| 图片模式 | 横版 16:9 封面（900×383px）；正文内图宽 900px，比例不限 |
| 发布要求 | 标题直接点明价值，不卖关子；正文结构清晰（小标题分段）；语气专业但不生硬；无需话题标签，结尾引导关注/转发 |
